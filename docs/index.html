Motus Game

Un jeu de lettres inspiré du célèbre jeu télévisé

À propos du projet

Motus Game est une implémentation en C du célèbre jeu de lettres Motus. Le joueur doit deviner un mot en un nombre limité d'essais, avec des indices visuels pour chaque tentative. Le jeu utilise SDL2 pour l'interface graphique et offre une expérience fluide et intuitive.
